1. Qual dos programas a seguir é o único
que não pode ser utilizado para escrever
códigos em texto sem formatação para
HTML + CSS + JS?

	Resposta: Word.

2. Qual é a empresa que fabrica o
VSCode, o editor que escolhemos para
escrever códigos para nosso curso?

	Resposta: Microsoft.

3. É possível usar o VSCode em idioma
Português do Brasil?

	Resposta: Sim. Mas não existe suporte nativo
para PT-BR, mas existe uma extensão
para isso.

4. Qual é a opção que devemos habilitar
nas configurações para que as instruções
muito longas sejam quebradas em várias
linhas automaticamente?

	Resposta: Word Wrap.

5. Qual é o caminho correto que
devemos seguir para ligar o salvamento
automático de arquivos no VSCode?

	Resposta: Arquivo > Salvar Automaticamente.

6. Qual é o nome do arquivo principal de
um site feito em HTML, que será aberto
assim que o visitante solicitar a abertura
do site?

	Resposta: index.html.

7. Ao criar um arquivo HTML, o VSCode
tem um atalho simples para criar o
código base para um arquivo desse tipo.
Que atalho é esse?

	Resposta: ! + Enter.

8. Qual é a tag HTML para criar uma
linha horizontal no corpo do navegador?

	Resposta: <hr>.

9. Para criar um parágrafo em HTML,
usamos a tag:

	Resposta: <p>.

10. A tag <title> é adicionada dentro
de qual área de um documento HTML?

	Resposta: <head>.
