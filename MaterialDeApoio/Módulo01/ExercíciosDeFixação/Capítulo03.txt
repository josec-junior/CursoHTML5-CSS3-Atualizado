1. Qual das frases a seguir é a única
tecnicamente CORRETA de se falar?
	
	Resposta: “Eu programo em linguagem JavaScript”.

2. A sigla HTML significa:

	Resposta: HyperText Markup Language.

3. A sigla CSS significa:

	Resposta:  Cascading Style Sheets.

4. Correlacione a coluna da esquerda
com a da direita, de acordo com o foco
de cada uma das tecnologias:

	Resposta: 3 - 1 - 2.

5. Qual tag abaixo não tem fechamento?

	Resposta: <meta>.

6. Na tag <a>, o href é um(a):
	
	Resposta: parâmetro.

7. Todas as configurações visuais dos
elementos HTML são realizadas pela
linguagem CSS. Agrupamos todas as
declarações CSS de um mesmo
elemento dentro de um(a):

	Resposta: seletor.

8. Para mudar a cor de um texto em
CSS, configuramos qual propriedade?

	Resposta: color.

9. Para indicar que um determinado
documento HTML está escrito na versão
mais recente da linguagem, devemos
adicionar a seguinte instrução:

	Resposta: <!DOCTYPE html>.

10. Qual é a tag de um documento HTML
adicionada pra manter a compatibilidade
com os caracteres acentuados, muito
comuns na língua Portuguesa?

	Resposta: <meta charset=“UTF-8”>.
